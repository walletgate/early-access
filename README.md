## What is WalletGate?

WalletGate provides a simple API for verifying EU Digital Identity Wallet credentials:

- **Age verification** — Verify users are 18+ without storing birthdates
- **EU residency checks** — Confirm EU/EEA residency for compliance
- **Identity verification** — Full KYC with government-issued credentials
- **Privacy-first** — Boolean results only, no PII storage

All built on real EU infrastructure (LOTL, OpenID4VP, ISO 18013-5).
